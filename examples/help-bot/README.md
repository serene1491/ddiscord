# Help Bot

This console focuses on the built-in help and error-behavior surfaces.

It demonstrates:

- built-in `help` pagination
- `@CommandCategory` and `@HideFromHelp`
- owner-only command behavior
- configurable command failure surfacing

## Run

```sh
cd examples/help-bot
dub run
```

Environment values come from `../.env` and `../.env.local`.
