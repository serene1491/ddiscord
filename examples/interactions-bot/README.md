# Interactions Bot

This console focuses on interactions, components, and modal flow.

It demonstrates:

- slash command sending a button panel
- message-component event handling
- opening a modal from a button
- modal submit handling with ephemeral response

## Run

```sh
cd examples/interactions-bot
dub run
```

Try:

- `/panel`
- click `Open Feedback Modal`
- submit the modal
- `/feedback` for direct modal open
